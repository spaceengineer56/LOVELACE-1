# Лавлейс-1  
### Бортовой компьютер ракеты «Заря SE-3»

**Лавлейс-1** — бортовой компьютер для ракеты «Заря SE-3», разработанной Space Engineer.  
Проект назван в честь Ады Лавлейс — первого программиста в истории.

Бортовой компьютер предназначен для:
- регистрации телеметрии во время полёта,
- определения апогея,
- активации системы спасения,
- записи данных на SD-карту,
- передачи координат посадки по SMS,
- звукового поиска после приземления.

---

## Возможности

- Запись телеметрии в CSV-формате:
  - номер кадра  
  - время (ms)  
  - температура  
  - давление  
  - широта  
  - долгота  
  - высота по GPS  
  - высота по барометру  

- Определение апогея по барометрической высоте  
- Активация системы раскрытия парашюта  
- Отправка координат через GSM (SIM800L)  
- Звуковая индикация после посадки  
- Логирование на SD-карту  

---

## Принцип работы

1. Считывается высота с BMP280.
2. Запоминается максимальная достигнутая высота.
3. Если снижение превышает 7 метров от апогея — активируется режим спуска:
   - включается система раскрытия парашюта;
   - каждые 15 секунд отправляется SMS с координатами;
   - включается звуковой сигнал для поиска.
4. Телеметрия записывается на SD-карту в файл `data.txt`.

---

## Используемые компоненты

- Arduino (совместимая плата)
- BMP280 (барометр)
- GPS-модуль
- SIM800L (GSM-модуль)
- SD-картридер
- Пьезоизлучатель
- Система раскрытия парашюта (исполнительный канал)

---

## Важно о SIM800L

В моём экземпляре SIM800L работал нестабильно.  
С высокой вероятностью это связано с браком конкретного модуля.

В наземных тестах:
- команды выполнялись корректно;
- SMS отправлялись;
- сбоев почти не наблюдалось.

Однако в реальном полёте модуль не проверялся.  
Гарантий корректного выполнения всех команд в полётных условиях нет.

## Структура данных (data.txt)

Файл записывается в формате CSV:


`frame,time,temp,pressure,lat,lon,gps_alt,bmp_alt`



Пример строки:

`15,12345,24.53,100845.32,55.123456,37.123456,152.34,148.92`
